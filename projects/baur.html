baur

Bubby’s AUR Helper

repo

About

Very simple AUR Helper. <100 lines of Rust code :D

Install

  1. Clone the repo: git clone https://github.com/BubbyRoosh1/baur
  2. Enter the directory: cd baur
  3. Build the project: cargo build --release
  4. Move the executable file into PATH: mv target/release/baur /usr/bin/

Use

Does everything you’d need in an AUR helper:

  • Query: -q
  • Install: -i
  • Clean (remove sources): -c
